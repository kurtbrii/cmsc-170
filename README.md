## Table of Contents

## Installation
1. Clone the repo: `git clone git@github.com:kurtbrii/cmsc-170.git`
2. Make sure [rye](https://rye.astral.sh/) is installed
3. `rye sync` to add the dependencies 

## Usage
Run the app with `rye run start`.
